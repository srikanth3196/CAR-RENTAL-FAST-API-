🚗 FastAPI Car Rental Service Backend

A fully functional backend system built using **FastAPI** that simulates a real-world car rental service. This project demonstrates API design, data validation, CRUD operations, and advanced backend features.

📌 Project Description
The Car Rental Service Backend allows users to:
- Browse available cars
- Register customers
- Book cars for rental
- Manage booking lifecycle (pickup & return)
- Search, sort, and paginate car listings

This project is designed demonstrate strong backend development skills using FastAPI.

🚀 Features

🔹 Core API Features
- RESTful API design using FastAPI
- Structured routing with proper endpoint hierarchy
- JSON-based request and response handling

🔹 Data Validation
- Input validation using **Pydantic**
- Field constraints (e.g., minimum values, required fields)
- Error handling with proper HTTP status codes

🔹 CRUD Operations
- Create, read, update, and delete car records
- Manage customer data

🔹 Multi-Step Workflow
- Booking → Pickup → Return
- Real-world rental lifecycle simulation

🔹 Advanced Functionalities
- 🔍 Search cars by brand and price
- 📊 Sorting cars by price (ascending/descending)
- 📄 Pagination support
- 🔗 Combined filtering (search + sort + pagination)
- 💰 Booking cost calculation
  
Tech Stack

| Technology | Usage |
